## Commands typed to generate this project

Open a terminal and type

```sh
mkdir using-express
cd using-express
npm init
npm install express
npm install --save-dev nodemon
```

## Running the project

```sh
node server.js

# if you don't want to kill the server and restart on every code change:
npx nodemon server.js
```
